User Manual
SPUDS IMS quick guide. Use Print User Manual to print this page.
1) Introduction
  • SPUDS IMS is a lightweight inventory and order tracking tool with modules for Vendors, Purchase Orders, Inventory, Sales Orders, Customers, and Settings.
  • Each module uses the same layout: Search/List on the left, record details on the right.
2) Features by Module
  • Dashboard: Quick counts and at-a-glance status.
  • Vendor: Search vendors and review vendor details.
  • Purchase Order: Search orders, view/edit lines, and track totals.
  • Inventory: Search products, manage locations/quantities, view vendors, BOM, and movement/history.
  • Sales Order: Search orders, view/edit lines, and track totals.
  • Customer: Search customers, edit extra info, and view order history.
  • Settings: Logo management and database maintenance tools.
3) Installation
  • Start the application using the provided Start/Stop scripts. The server will display accessible URLs.
  • Open the app in a browser using the URL from the server output (example: http://localhost:3200/).
  • Log in using the configured credentials (if enabled).
4) User Instructions
  • Navigation: Use the top tabs to switch modules.
  • Search: Use the left-side filters in each module to find records quickly.
  • Edit: Select a record from the list, update fields on the right, then click Save where available.
  • Items: For Purchase Orders and Sales Orders, use Add Row / Delete Row under Items to manage lines.
  • Settings: Use Backup/Restore for database files; use maintenance tools only when needed.
5) Troubleshooting
  • Cannot open the app: Ensure the server is running and use the correct URL shown in the terminal.
  • Database connection failed: Start MariaDB and verify the DB port is reachable.
  • Restore problems or strange characters: Run Normalize Collations, then try again.
  • Duplicate rows: Use Fix Duplicate Records (keeps one copy of exact duplicates).
  • Admin tools rejected: Ensure the admin password is configured and you are logged in.
